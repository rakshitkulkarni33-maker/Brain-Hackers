# 🏥 Serenity Healthcare

A modern **Doctor Availability & Teleconsultation Platform** that allows users to connect with doctors online بسهولة.

---

## 🚀 Features

* 👨‍⚕️ Doctor listing & availability
* 📅 Appointment booking
* 🎥 Video consultations
* 💊 Digital prescriptions
* 📂 Medical records
* ⏰ 24/7 access
* 📱 Responsive design

---

## 📁 Project Files

* `index.html` – Homepage 
* `doctors.html` – Doctors list 
* `scheduling.html` – Booking system 
* `video-calls.html` – Consultation 
* `medical-records.html` – Records 
* `digital-prescriptions.html` – Prescriptions 
* `login.html` / `register.html` – Auth pages

---

## 🛠️ Tech Stack

* HTML, CSS, JavaScript
* Font Awesome
* Google Fonts

---

## ▶️ How to Run

1. Download or clone repo
2. Open `index.html` in browser
